网页制作小知识

  1. 关于导航栏：让导航栏横过来（横向结构）（浮动？）
    当页面出现布局bug，就使用border大法！
    border: 1px solid red; 1像素的实心的红色边框
    在li标签中加float标签，在ul标签中加clearfix，并且在css中添加三句代码：
    首先在所有的子元素上加float：left，在这些子元素的爸爸加class=“clearfix”，并且在css中添加三句代码：

clearfix写法很多，就背这个就好，这个必须会背，就可以解决页面布局问题。

  1. 如果去除列表前面的黑点：在ul中加入 style="list-style: none;"

  2. 如何去除列表的默认样式：在ul中加入 style="margin: 0;padding: 0;"

  3. 选择器：> 表示儿子

  4. 开发者工具： styles 表示样式； computed 表示计算出来的样式： 考点！！！

  5. font-size: 12px; px是像素
  6. 常识： 不同操作系统对相同颜色的字的渲染是不一样的，会出现虚边，直接用取色工具会不准确。
    a.直接抄源代码上写的颜色
    b.问设计师
  7. 去除下划线：text-decoration: none; decoration: 装饰
  8. 如果不知道一个东西的取值：
    a.查MDN
    b.直接在开发者工具里对应位置styles中修改代码；各种样式都有，可以试出来
  9. 修改间距：margin-left: xxpx; margin-right: xxpx; 经验之谈：写在li里
  10. 加粗 font-weight: bold;

  11. 鼠标悬停出现边框：在对应位置比如： .topNavBar>nav>ul>li>a是对应位置，那就再写一个

  12. 解决鼠悬停时位置发生变化：因为出现了边框占用位置所以才移动，所以干脆在原本上加上透明的边框，当当当！ border-bottom: 3px solid transparent; transparent 透明

  13. 把边框只保留下面就像下划线一样：把border 变成 border-bottom 。 bottom：底部

  14. 强制触发悬停特效：开发者工具-点击对应位置-点hov-点：hover 即可，被强制触发悬停特效的代码左边会有个黄色的圆
  15. padding 内边距: padding-top padding-bottom
    border 边框
    margin 外边框
  16. 如果a标签比li标签高，就是li没有包住a
    解决方法：在a里面写 display:block;
    所以没有什么内联元素和块级元素；所有的元素既可以是内联元素也可以是块级元素。

  17. 去除a标签的蓝色：

    div{

    border: 10px solid transparent;
width: 0px;
border-left-color: #E6686A;
border-top-width: 0px;

    }

    .userCard svg{
    width: 30px;
    height: 30px;
    fill: white;
    vertical-align: top; /使上下空隙一样 （先记忆）/
    }
    `

  18. 让三个图标居中 再三个a标签的父元素media里写 text-align: center;

  19. 用圆圈住图标：
    1.把矩形变成圆 border-radius: 50%;
    2.让圆变大
    width: 40px;
    /*height: 40px;*/
    line-height: 30px;
    3.让图标居中 因为写了heigh ，所以出bug，这里就不写高度，我们直接用padding代替,因为行高是30px，所以padding上下是5px即可达到40px：padding: 5px 0;

  20. 最后再调整一下高度就好

  21. CSS position 5个权值：sticky fixed absolute relative static

  22. CSS 中 width 的默认值是auto 即自动调节

  23. content-box 与 border-box 的区别是： content-box 的 width 不包括 padding 和 border
    理论上说，content-box与border-box区别是两者的盒子的宽度是否包含表框和内边距
